# Horror Maze

A web horror maze created in HTML, CSS & JavaScript for a game-jam.

## Run Locally

Clone the project

```bash
  git clone https://github.com/nhuijser/horror-maze
```

Start the server using the Live Server extension or anything of your choice.

## Features

- Horror themed
- Mobile Support
